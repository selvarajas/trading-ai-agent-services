# create a new repository on the command line
-----------------------------------------------------
## Git Init Command
-----------------------------------------------------
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/selvarajas/trading-ai-agent-services.git
git push -u origin main
